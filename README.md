# 42 Piscine Python — Exercises

This repository contains my **Python exercises for the 42 curriculum**, organized by module folders from `Python-00` to `Python-10`.

## Structure

- `Python-00/` … `Python-10/`  
  Each folder corresponds to a module/day of the 42 Python curriculum and contains the required exercises (and any supporting files).

## Requirements

- **Python 3.x** (recommended: 3.10+)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/MAN3RAF/Python.git
   cd Python
   ```

2. Go to a module folder and run an exercise:
   ```bash
   cd Python-00
   python <exercise_file>.py
   ```

> Some exercises may require arguments or interactive input depending on the subject.
